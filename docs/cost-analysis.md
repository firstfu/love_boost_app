## 1. 💰 API成本分析

### 1.1 OpenAI API定價 (2025年最新)

#### 🤖 GPT-4o (主要文字分析)
- **輸入Token**：$5.00/百萬Token
- **輸出Token**：$15.00/百萬Token  
- **換算**：1個字≈1.4 Token，1000字≈1400 Token

#### 👁️ GPT-4o Vision (圖片分析)
- **低解析度**：85 Token/張 ≈ $0.0004/張
- **高解析度**：1100 Token/張 ≈ $0.0055/張
- **平均成本**：$0.003/張（中解析度）

#### 💡 GPT-4o Mini (節省選項)
- **輸入Token**：$0.15/百萬Token
- **輸出Token**：$0.60/百萬Token
- **適用場景**：簡單分析，準確度90%

### 1.2 每用戶API成本計算

#### 📊 基礎用戶使用情境
| 使用項目 | 數量 | 說明 |

**1. 圖片分析成本**
```
5人 × 3張照片 × $0.003/張 = $0.045/月
```

```

小計：$0.029/月
```

**3. 對話建議成本**
```
輸入：150次 × 50字提示 × 1.4Token × $5/百萬Token = $0.053/月
輸出：150次 × 200字建議 × 1.4Token × $15/百萬Token = $0.630/月
```
