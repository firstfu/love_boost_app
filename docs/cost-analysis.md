## 1. 💰 API 成本分析

### 1.1 OpenAI API 定價 (2025 年最新)

#### 🤖 GPT-4o Mini (統一模型 - 多模態)

- **輸入 Token**：$0.15/百萬 Token
- **輸出 Token**：$0.60/百萬 Token
- **多模態支援**：文字+圖片分析
- **換算**：1 個字 ≈1.4 Token，1000 字 ≈1400 Token

#### 👁️ GPT-4o Mini Vision (圖片分析)

- **低解析度**：85 Token/張 ≈ $0.0000128/張
- **高解析度**：1100 Token/張 ≈ $0.000165/張
- **平均成本**：$0.0001/張（中解析度，約 400 Token）

#### 💡 成本優勢

- **統一模型架構**：單一 GPT-4o Mini 處理所有任務
- **多模態整合**：文字+圖片分析無縫整合
- **效能表現**：接近 GPT-4o 的 90%準確度

### 1.2 每用戶 API 成本計算

#### 📊 基礎用戶使用情境

| 使用項目 | 數量       | 說明                  |
| -------- | ---------- | --------------------- |
| 目標對象 | 5 人/月    | 分析不同異性          |
| 照片分析 | 15 張/月   | 每人 3 張中解析度照片 |
| 聊天記錄 | 2500 字/月 | 每人 500 字對話記錄   |
| 對話建議 | 150 次/月  | 每天約 5 次建議請求   |

#### 💵 詳細成本拆解

**1. 圖片分析成本 (GPT-4o Mini Vision)**

```
5人 × 3張照片 × $0.0001/張 = $0.0015/月
```

**2. 文字分析成本 (GPT-4o Mini 統一模型)**

```
輸入：5人 × 500字 × 1.4Token × $0.15/百萬Token = $0.000525/月
輸出：5人 × 100字報告 × 1.4Token × $0.60/百萬Token = $0.00042/月
小計：$0.000945/月
```

**3. 對話建議成本 (GPT-4o Mini 統一模型)**

```
輸入：150次 × 50字提示 × 1.4Token × $0.15/百萬Token = $0.00158/月
輸出：150次 × 200字建議 × 1.4Token × $0.60/百萬Token = $0.0252/月
小計：$0.0268/月
```

#### 🏷️ 成本總結 (統一使用 GPT-4o Mini)

| 用戶類型     | 月 API 成本 | 新台幣 |
| ------------ | ----------- | ------ |
| **基礎用戶** | $0.029      | NT$0.9 |
| **重度用戶** | $0.087      | NT$2.7 |
| **平均用戶** | $0.058      | NT$1.8 |

> 💡 **統一模型優勢**：使用單一 GPT-4o Mini 模型，簡化架構且成本極低！
